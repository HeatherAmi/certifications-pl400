# Contributing

Thanks for your interest in contributing.

This repo is primarily a learning resource that follows my YouTube series. Pull requests are welcome, especially for:

- Fixing typos or broken links
- Clarifying unclear instructions
- Improving accessibility of guides
- Suggesting better ways to explain a concept

## Before opening a pull request

1. Open an issue first so we can discuss the change
2. Keep changes scoped to one thing per pull request
3. Use UK English spelling
4. Keep the tone professional but conversational
5. No em dashes in written content (style choice for consistency)

## What I will not accept

- Major restructuring without prior discussion
- Removal of educational scaffolding (such as the trainee-guide.md files)
- Changes that conflict with my recorded YouTube videos
