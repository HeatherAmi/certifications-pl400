# Project 04: Form Validation with JavaScript

A client-side script that validates a model-driven form on save, with friendly error messages and field-level highlights.

## Status

Planned.

## Skills covered

- JavaScript web resources
- Form event handlers (OnLoad, OnSave, OnChange)
- The form context API
- Xrm.WebApi for lookups
- Notifications and field-level errors

## Estimated time

Around 45 minutes.
