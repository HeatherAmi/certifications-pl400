# Project 01: Hello Dataverse

A short, gentle introduction to Dataverse, solutions, and model-driven apps. This is the foundation every other project in this series builds on.

## Status

In progress. Content coming soon.

## What you will build

A custom table, a few columns, a basic form and view, and a simple model-driven app, all packaged inside a proper Dataverse solution.

## Skills covered

- Creating a publisher and solution
- Adding a custom table
- Adding and configuring columns
- Creating a model-driven app
- Adding a business rule
- Exporting a solution

## Estimated time

Around 30 minutes.
