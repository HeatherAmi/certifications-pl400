# Project 09: Service Bus Integration

An asynchronous Dataverse plug-in that posts a message to an Azure Service Bus queue, consumed by a Function App that writes results back to Dataverse.

## Status

Planned.

## Skills covered

- Azure Service Bus queues
- Service Bus plug-in registration
- Asynchronous integration patterns
- Function Apps with Service Bus triggers
