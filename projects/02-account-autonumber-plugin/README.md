# Project 02: Account Auto-Numbering Plug-in

A synchronous Dataverse plug-in that automatically generates a unique reference number when an Account record is created.

## Status

Planned.

## What you will build

A C# class library, registered as a synchronous plug-in on the Create message of the Account table, that writes a generated reference number into a custom column.

## Skills covered

- Writing a plug-in in C# against the Dataverse Organization Service
- Registering a plug-in with the Plugin Registration Tool
- Using the tracing service for diagnostics
- Working with plug-in execution context

## Estimated time

Around 45 minutes.
